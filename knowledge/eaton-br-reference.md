# Eaton BR Breaker Series — Reference Guide

Grizzly Electrical Solutions standard panel brand. All new panel installs default to Eaton BR
unless the customer already has a CH or Square D panel and a full replacement isn't in scope.

---

## Breaker Type Codes

| Code | Type | Notes |
|------|------|-------|
| BR | Standard single-pole or double-pole | Most common; no AFCI/GFCI |
| BRP | Plug-on neutral (PoN) version | Used in newer PoN-ready enclosures; neutral clips in instead of pigtail |
| BRN | BR with neutral pigtail | For circuits needing switched neutral |
| BRD | Tandem (duplex) breaker | Two circuits in one slot; not compatible in all panels — check tandem chart |
| BRAF / BRCAF | AFCI (Arc Fault Circuit Interrupter) | Required for bedrooms, living rooms, dining under NEC 2020 |
| BRGF / BRCGF | GFCI (Ground Fault Circuit Interrupter) | Required within 6 ft of water, outdoor, garage, unfinished basement |
| BR_DF | Dual Function (AFCI + GFCI combined) | e.g., BR120DF; meets both requirements in one device |
| BRHH | High interrupting capacity | For HID lighting, motor loads; higher kAIC rating |

> **CH vs BR**: NOT interchangeable. CH uses a different clip geometry — CH breakers will seat
> loosely in a BR panel and will not make proper bus contact. Never mix.

> **Cutler-Hammer = Eaton**: Same product, rebranded. CH and BR lines from Cutler-Hammer are
> identical to current Eaton. Part numbers are the same.

## Panel Compatibility Matrix

### BR100 (100A main lug or main breaker, 20-slot)
- Accepts: BR, BRN, BR_DF, BRAF, BRGF standard single and double-pole
- Tandem (BRD/BRKR): allowed in slots 3–20 only (up to 10 tandems)
- Full-size slots: 20 (= 20 single-pole or 10 double-pole circuits)

### BR200 (200A main breaker, 40-slot) — most common Grizzly install
- Accepts: all BR series including PoN variants
- Tandem: allowed in slots 3–40; up to 20 tandems (check label inside door for exact limits)
- Full-size slots: 40

### BR2040 (200A, 40-circuit enclosure — combo panel)
- Same breaker compatibility as BR200
- Often shipped with a 200A main breaker pre-installed
- Full-size slots: 40

### BR1224 (125A, 24-slot — smaller residential or garage sub)
- Full-size slots: 24
- Tandem allowed in lower half (slots 13–24)

### BRP Series (Plug-on Neutral enclosures)
- Requires BRP-suffix breakers (e.g., BRP115, BRP120DF) — standard BR breakers will fit
  mechanically but won't make the neutral connection
- Look for "PoN" or "Plug-on Neutral" label on enclosure

---

## Tandem (Duplex) Breakers — Rules

- Tandems put two 1-pole circuits in one physical slot
- Only use where the panel label explicitly permits ("tandem-capable" slots)
- Grizzly rule: always check the slot map printed inside the panel door before installing a tandem
- Never install a tandem in a slot not rated for it — bus connection is not guaranteed
- BRKR tandems are NOT AFCI/GFCI rated — if the circuit requires AFCI/GFCI, use a full-size
  AFCI/GFCI breaker in a standard slot or add a GFCI outlet at first device

---

## BR vs CH — Do Not Mix

| Feature | BR Series | CH Series |
|---------|-----------|-----------|
| Bus clip geometry | BR clip (wider) | CH clip (narrower) |
| Interchangeable? | NO | NO |
| Common in TX installs | Yes — default | Yes — older homes |
| Can CH breaker fit BR panel? | Seats but won't latch correctly — NEVER do this |

If a customer has an existing CH panel, order CH breakers (CHF115, CHF120, etc.), not BR.
Mixing series is a code violation and a fire hazard.
